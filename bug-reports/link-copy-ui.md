# Bug: Copy Link Feedback Missing & UI Layout Issue

## Module
Event Details Page

## Description
- "Copy Link" action does not provide any user feedback  
- Layout has excessive whitespace on the right side  
- "View Terms & Conditions" link is not working  

---

## Steps to Reproduce
1. Open Event Details page  
2. Click on "Copy Link"  
3. Click on "View Terms & Conditions"  

---

## Expected Result
- Copy action should show confirmation (e.g., "Copied!")  
- T&C link should open properly  
- Layout should be balanced  

---

## Actual Result
- No feedback after copying link  
- T&C link does not respond  
- UI has large empty space  

---

## Severity
Medium
