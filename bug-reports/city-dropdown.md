# Bug: City Dropdown Not Populating for Selected State

## Module
Event Management → Create Event

## Description
When selecting a specific state, the city dropdown does not populate any values. Since the city field is mandatory, the user is unable to proceed with event creation.

---

## Steps to Reproduce
1. Navigate to Create Event form  
2. Select State (e.g., New Brunswick)  
3. Open City dropdown  

---

## Expected Result
- City dropdown should display available cities based on selected state  
- User should be able to select a city and proceed  

---

## Actual Result
- City dropdown is empty  
- User cannot complete event creation  

---

## Severity
High
