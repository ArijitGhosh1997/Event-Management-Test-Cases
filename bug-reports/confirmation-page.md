# Bug: Confirmation Page Content Not Rendering Dynamically

## Module
Event Registration Confirmation Page

## Description
The confirmation page does not display dynamic event-specific details. It only shows a static message, which creates confusion for users.

---

## Steps to Reproduce
1. Complete event registration  
2. Navigate to confirmation page  

---

## Expected Result
- Event details (name, date, location, description) should be displayed  
- User should clearly know which event they registered for  

---

## Actual Result
- Only static confirmation message is shown  
- Event details are missing  

---

## Severity
Medium
