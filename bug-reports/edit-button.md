# Bug: Edit Button Not Working in Configuration Management

## Module
Configuration Management

## Description
The Edit button for configuration items (such as badge categories or user types) is not working. Clicking the button does not trigger any action.

---

## Steps to Reproduce
1. Navigate to Configuration Management  
2. Click on Edit button for any item  

---

## Expected Result
- Edit modal or form should open  
- User should be able to update details  

---

## Actual Result
- Clicking Edit button does nothing  
- No modal or action is triggered  

---

## Severity
Medium
