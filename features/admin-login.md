# Feature: Admin Login with Secure Authentication

## Description
Admin users should be able to log in securely and access the dashboard based on their role.

---

## Acceptance Criteria
- Valid credentials → login successful  
- Invalid credentials → error message  
- Non-admin users → access denied  
- Session should remain active until logout or expiry  

---

## User Flow
1. Enter email and password  
2. Click login  
3. System authenticates user  
4. Redirect to dashboard if valid  

---

## Type
Feature / User Story
