# Enhancement: URL Validation Improvement

## Description
The system should validate URLs properly and accept only valid formats.

---

## Requirements
- Accept only http and https URLs  
- Must include scheme and host  

---

## Valid Examples
- https://example.com  
- https://example.com/path  

---

## Invalid Examples
- example.com  
- https:///path  

---

## Expected Result
- Invalid URLs should show error message  
- Valid URLs should be accepted  

---

## Type
Enhancement
